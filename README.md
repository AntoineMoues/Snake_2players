# [Snake 2-players]()

A 2-players snake created for class

## Versions

- [1 player](https://antoinemoues.github.io/snake/)
- [Online-2 players version(WIP)]()

## Controls

Blue Snake:
  - Z : Up
  - Q : Left
  - S : Down
  - D : Right

Red Snake:
  - I : Up
  - J : Left
  - K : Down
  - L : Right
